## Why I Built It

Little Broomstick Tales is a small game project I started for my daughter. The premise is deliberately simple: guide a little witch cat around a forest, make the controls approachable, and let the project be more about charm than scale.

It is also a good excuse to work in a different mode. Games force decisions about motion, input, feedback, and delight that normal web apps can avoid.

## How It Works

The project uses Phaser 3 with Vite and TypeScript. The game code is organized into scenes, reusable objects, and shared utilities. It supports keyboard, gamepad, and touch controls, which keeps the interaction model flexible across laptop and tablet-style play.

The code started from a Phaser template, but the useful work is in shaping that starter into a small game with a real audience and a coherent feel.

## What I Like About It

The best part is that the project has a clear emotional reason to exist. It is not a portfolio demo pretending to be a game. It is a small game for someone specific.
